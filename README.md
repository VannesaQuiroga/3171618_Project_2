#  Agencia de Turismo - CRUD con React + TypeScript

##  Descripción
Aplicación web para gestionar paquetes turísticos.
Permite agregar, editar, eliminar y visualizar paquetes.

##  Tecnologías utilizadas
- React
- TypeScript
- Vite

##  Estructura
- types → Definición de interfaces
- components → Componentes reutilizables
- App.tsx → Lógica principal

##  Funcionalidades
- Crear paquete turístico
- Editar paquete
- Eliminar paquete
- Validaciones básicas
- Renderizado condicional

##  Conceptos aplicados
- useState
- useEffect
- Props tipadas
- Interfaces
- Spread operator
- map y filter

## Ejecutar
- pnpm install
- pnpm run dev
